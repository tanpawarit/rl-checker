Implementation

  • Structure: app/web/routes.py (2 routes: GET / form · POST /check-ui calls run_check → renders a partial) + app/web/templates/ + app/web/static/ — the brain is untouched
  • Every field maps directly to the schema: the verdict is counted from findings[].status · section 01 = saw · flags = flags · the card orders evidence → issue → fix_note, following the field order in schemas.py
  • The real logo is embedded as a data URI in this file (the live screen uses /static/logo-nhj.svg) · fonts load from Google Fonts in the real template
  • Light theme only, by decision (18 Jul 2026) — the dark variant was removed from both this mock and the live app; don't add it back without discussing
